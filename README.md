# Insurance Data Analysis Dashboard

## Objective
The goal of this project is to analyze data from an insurance company to understand performance across cities, insurance types, and sales channels.  
The analysis helps identify which areas generate the most profit and where improvements are needed.

## Key Questions
1. Which cities have the highest profit?
2. Which type of insurance is the most profitable?
3. Which sales channel performs best?
4. Which insurance type has the most claims?

## Main Insights
- Dammam and Abha have the highest profit, followed by Riyadh.
- Car insurance gives the most profit but also has the most claims.
- The Agent channel performs better than Online and Branch channels.
- Home insurance has the lowest claims and the lowest profit.

## Recommendations
- Focus on car insurance but try to control claim costs.
- Support and improve the agent sales channel.
- Review pricing and marketing in Jeddah and Makkah.
- Try to improve online sales performance.
